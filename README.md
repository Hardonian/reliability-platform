# Reliability Platform

**Ops automation as a service for startups.**

## Features

- GitHub webhook integration
- AI-powered incident analysis
- Automated fix recommendations
- SLA monitoring
- Team alerts

## Quick Start

```bash
npm install
npm run dev
```

## Webhook Setup

Point your monitoring tool webhooks to `/webhook`

## Tech Stack

- Node.js + Express
- Prisma
- Claude API
- GitHub webhooks
